## 버블 정렬 (Bubble Sort)

---

### 버블 정렬 알고리즘 개념 요약

* 서로 **인접한** 두 원소를 검사하여 정렬하는 알고리즘
* 선택 정렬과 기본 개념이 유사하다.



### 소스 코드

```c++
// bubble sort
for (int i = n-1; i > 0; i--) {
    for (int j = 0; j < i; j++) {
        if (arr[j] > arr[j+1]) {
            swap(arr[j], arr[j+1]);
        }
    }
}
```
