## Port & Port Number

<br>

### :book: Port

운영 체제 통신의 종단점이다. 소프트웨어에서는 `네트워크 서비스`나 `특정 프로세스`를 식별하는 논리 단위이다. 즉, IP 내에서 Port 번호로 애플리케이션을 구분한다.

<br>

### :book: Port 번호

* Port 번호의 범위

포트번호는 0 ~ 65535번까지 사용할 수 있고, `Well-known Ports`, `Registered Ports`, `Dynamic Ports`의 세 종류로 구분된다.
