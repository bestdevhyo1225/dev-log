## .jar 에 대해서 알아봅시다..

<br>

### :book: Java Archive File (자바 아카이브 파일)

* 여러 개의 Java 클래스 파일과 클래스들이 이용하는 관련 리소스(텍스트, 이미지 등) 및 메타데이터를 하나로 모이서 자바 플랫폼에 응용 소프트웨어나 라이브러리를 배포하기 위한 소프트웨어 패키지 파일 포맷

* 실제로 Zip 파일 포맷으로 이루어진 압축 파일이며, 확장자는 `.jar`이다.
